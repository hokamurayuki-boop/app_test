夜勤手当計算アプリ

訪問介護シフトCSVから夜勤手当を自動計算します。

🔒 ファイルはブラウザ内でのみ処理され、外部サーバーには送信されません。

STEP 1: CSVファイルを選択

STEP 2: 対象年月の確認

ファイル名から自動抽出します。誤りがあれば修正してください。

STEP 3: 計算結果

計算ルール

  • 夜勤時間帯: 22:00 〜 翌8:00
  • 単価: 500円 / 30分（30分未満は切り上げ: 10分→500円、40分→1000円）
  • 1シフトあたりの上限: 5,000円
  • シフト境界: 同一従業員で「前行終了時刻 = 次行開始時刻」の行を1シフトとして扱う
  • 上限超過時: 先頭行から順に充当（FIFO方式）

CSVフォーマット（Shift-JIS / 10列）

ヘルパー名 / 日付 / 曜日 / 利用者 / 業務種別 / サービス内容 / 開始時間 / 終了時間 / 提供時間 / 備考

動作確認（組み込みテスト）

計算ロジックを仕様テストで検証します。
